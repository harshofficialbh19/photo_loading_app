# Photo App

This project is an Android application developed using Kotlin and Jetpack Compose to display photos fetched from a remote API and stored locally in a database.

## Prerequisites

Before running the code, make sure you have the following installed on your system:

- Android Studio Iguana | 2023.2.1 Patch 2
- Gradle version 8.7
- Android Gradle plugin version 8.0.2

## Getting Started

Follow these steps to run the code in Android Studio:

1. Clone the repository to your local machine:

2. Open Android Studio.

3. Click on "Open an existing Android Studio project" and select the cloned repository directory.

4. Once the project is opened, wait for Android Studio to sync the project and download any necessary dependencies.

5. Connect a physical device or set up an emulator through the AVD Manager in Android Studio.

6. Click on the "Run" button (green triangle) in the toolbar, or select "Run > Run 'app'" from the menu.

7. Choose your target device (physical device or emulator) and click "OK" to run the application.

8. Wait for the app to build and deploy to the selected device. Once deployed, you should see the app running on your device.

## Troubleshooting

If you encounter any issues while running the code, try the following troubleshooting steps:

- Make sure your Android Studio version matches the specified version in the prerequisites.
- Check that Gradle is properly installed and configured in your Android Studio settings.
- Ensure that your Android SDK is up-to-date and compatible with the project requirements.
- If you encounter any build errors, try cleaning and rebuilding the project by selecting "Build > Clean Project" and "Build > Rebuild Project" from the menu.
